# Spatial Omics Tumor Drug Response Prediction

This Jupyter Book is a tracked reporting interface for the repository. It helps a teacher or mentor inspect the project from a fresh GitHub checkout without relying on ignored local folders, private data, generated model artifacts, or long-running workflows.

The notebooks summarize tracked project files, repository structure, pipeline architecture, current documentation, and recent Git history. They do not run production pipelines, train models, download public data, validate external URLs, or execute inference.
