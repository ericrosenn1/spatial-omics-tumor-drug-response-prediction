# Spatial Omics Tumor Drug Response Prediction

This Jupyter Book is a tracked reporting and code-evolution interface for the repository. It helps Prof. Mgavi or another independent-study mentor inspect the project from a fresh GitHub checkout without relying on ignored local folders, private data, generated model artifacts, or long-running workflows.

The notebooks summarize tracked project files, repository structure, pipeline architecture, current documentation, recent Git history, and selected source-code organization. They do not run production pipelines, train models, download public data, validate external URLs, or execute inference.
