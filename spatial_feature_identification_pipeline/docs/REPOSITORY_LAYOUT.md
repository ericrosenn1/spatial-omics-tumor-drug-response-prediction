# Repository layout

code/ contains active source scripts and reusable library modules.
configs/ contains YAML configuration files.
	ools/ contains helper utilities that do not define the scientific pipeline.
docs/ contains project status, run instructions, provenance notes, and reports.
outputs/ contains generated output folders and is ignored by git by default.
_repo_local_archive/ contains local backup and audit material and is ignored by git.
